### Uncharted Tonemapping, applies only when "tonemap_function" is set to "Uncharted"
### A = shoulder_strength
### B = linear_strength
### C = linear_angle
### D = toe_strength
### E = toe_numerator
### F = toe_denominator
### F(x) = ((x(A*x + C*B) + D*E) / (x(A*x + B) + D*F)) - E/F
### FinalColor = F(LinearColor)/F(LinearWhite)
